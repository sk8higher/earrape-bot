# earrape-bot

Telegram bot for making loud audios, written in Ruby.

[Bot link (hosted on free Railway tier, may be unactive)](https://t.me/earrapemaker_bot)

What I've used:

- Ruby 3.2.2
- telegram-bot-ruby
- ffmpeg to process audio
- Sidekiq for handling audio processing tasks
